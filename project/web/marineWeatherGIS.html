WEB DESIGN

국립해양측위정보원 해양기상 신호표지 모니터링 (PC)

한 화면 내에서 모든 정보가 확인되어야 하는 GIS 시스템의 특성을 고려해 진행한 웹 화면 개선 프로젝트입니다.서브페이지 이동 없이 정보가 집약되는 구조로, 지도와 데이터가 동시에 인지될 수 있도록 화면 구성을 정리했습니다. 개선 방향에 따라 두 가지 화면(ver.01 / ver.02)을 제작하며, 정보 배치와 시각적 우선순위를 비교·정리한 작업입니다.
